# 会議室の詳細を確認する - 利用者・オーナー向けフロントエンド仕様

## 変更概要

会議室詳細画面を実装する。会議室種別（物理/バーチャル）に応じて表示セクションを動的に切り替え、評価スコア・レビュー・運用ルール・予約ボタンを提供する。

## 画面仕様

### 会議室詳細画面

- **URL**: `/rooms/{room_id}`
- **アクセス権**: 全ユーザー（未ログインも可）
- **ポータル**: user

#### 表示要素とコンポーネントマッピング

| 要素 | 種別 | デザインシステムコンポーネント | 説明 |
|------|------|------------------------------|------|
| 会議室画像カルーセル | 画像 | - (カルーセルUI) | 複数画像のスライド表示（CDN 配信） |
| 会議室種別バッジ | バッジ | Badge (variant=info/virtual) | 物理=Blue、バーチャル=Violet |
| 会議室名・価格 | テキスト | PriceDisplay | 会議室名（h1）・時間単価（PriceDisplay） |
| 評価スコアブロック | 評価 | StarRating (readonly=true) | 平均スコア・レビュー件数 |
| 設備・機能タグ | バッジ | Badge (variant=outline) | 設備一覧をタグ形式で表示 |
| 会議室情報カード（物理） | カード | Card | 所在地・広さ・収容人数（物理のみ） |
| バーチャル情報カード | カード | Card | 会議ツール種別・同時接続数・録画可否（バーチャルのみ） |
| 運用ルールカード | カード | Card | 利用可能時間帯・最低/最大利用時間・貸出可否 |
| レビュー一覧 | リスト | Card × 10件 | 評価スコア・コメント・評価日時 |
| 予約ボタン | ボタン | Button (variant=primary, size=lg) | 「予約する」（ログイン必須） |
| オーナー問合せボタン | ボタン | Button (variant=outline) | 「オーナーに問合せる」 |
| オーナー認証バッジ | バッジ | OwnerVerificationBadge | オーナーの認証済み表示 |

| 背景色 | var(--semantic-background) | #FFFFFF |
| アクセント | var(--portal-primary) | #2563EB（Blue 600） |

| 評価スター色 | var(--semantic-rating) | #FBBF24 |

#### UIロジック

- **状態管理**: 会議室種別（physical/virtual）によるセクション表示切替
- **バリデーション**: なし（読み取り専用画面）
- **ローディング**: 画面初期表示時にスケルトンUI表示
- **エラーハンドリング**: 404 の場合「会議室が見つかりません」エラーページに遷移
- **未ログイン時の予約ボタン**: クリックでログイン画面にリダイレクト（リダイレクト先 URL を保持）

#### 操作フロー

1. 会議室一覧から会議室カードをクリック
2. 会議室詳細画面に遷移（/rooms/{room_id}）
3. 画像カルーセル・会議室情報・運用ルール・評価が表示される
4. 「予約する」ボタンをクリック → 予約申請画面（物理）またはバーチャル会議室予約画面（バーチャル）へ遷移
5. 「オーナーに問合せる」ボタンをクリック → オーナー問合せ画面へ遷移

## コンポーネント設計

### RoomDetailHeader

- **ベースコンポーネント**: RoomCard（詳細拡張）
- **Props**:
  | Prop | 型 | 必須 | 説明 |
  |------|---|------|------|
  | room | RoomDetail | Yes | 会議室詳細情報 |
  | onBookingClick | () => void | Yes | 予約ボタンクリック |
  | onInquiryClick | () => void | Yes | 問合せボタンクリック |
- **状態**: なし
- **イベント**: onBookingClick、onInquiryClick

### RoomReviewList

- **ベースコンポーネント**: Card
- **Props**:
  | Prop | 型 | 必須 | 説明 |
  |------|---|------|------|
  | reviews | Review[] | Yes | レビュー一覧（最新10件） |
  | avgRating | number | Yes | 平均評価スコア |
  | totalCount | number | Yes | レビュー総件数 |
- **状態**: なし
- **イベント**: なし
