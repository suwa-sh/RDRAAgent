## チャート選定ガイドライン

### 観点別チャート選定

| 観点 | 推奨チャート | 使用場面 |
|------|-----------|---------|
| Comparison（比較） | 横棒グラフ | 会議室別売上/利用回数の比較。ラベルが長い場合に横棒が有効 |
| Composition（構成比） | ドーナツチャート（4項目以下） | 売上分析区分別構成比、会議室種別比率。中央にKPI数値を配置 |
| Distribution（分布） | ヒートマップ | 時間帯 x 曜日の稼働率。色の濃淡で密度を表現 |
| Trend（傾向） | 折れ線グラフ | 月別推移。複数系列は3本まで。エリアチャートは重なりが少ない場合のみ |

### データ表現フレームワーク

| 表現方法 | 用途 | 適用画面 |
|---------|------|---------|
| % Change（変化率） | 前月比・前年比の成長率 | 手数料売上分析、利用状況分析 |
| Variance（差異） | 目標売上との差分 | 売上実績確認 |
| Calculated Metric（計算指標） | 会議室あたり平均売上、利用者あたり利用回数 | 利用状況分析、売上実績確認 |
| Added Context（追加コンテキスト） | 前年同月の折れ線を薄い色で重ねる | 手数料売上分析、利用回数確認 |

## ダッシュボード設計原則

### 管理者ダッシュボード（手数料売上分析 + 利用状況分析）

#### 情報の階層化

- **全体サマリー（L1）**: 今月の手数料売上合計、利用回数合計、アクティブオーナー数、アクティブ利用者数（4 KPI カード）
- **ドリルダウン（L2）**: 月別推移チャート + 会議室別ランキング。サマリーカードをクリックで該当チャートへスクロール
- **詳細データ（L3）**: テーブルでの個別データ表示。CSVエクスポート対応

#### データストーリーテリング

- **ナラティブ**: 「先月比で手数料売上が12%増加。バーチャル会議室の利用が牽引」等のサマリーテキストを自動生成
- **比較軸**: 前月比（デフォルト）、前年同月比（切替可能）
- **アクション**: 売上低下 → 対象会議室の詳細リンク。利用減少 → 利用者動向の分析リンク

### オーナーダッシュボード（売上実績 + 利用回数）

#### 情報の階層化

- **全体サマリー（L1）**: 今月売上合計、利用回数合計、平均評価スコア（3 KPI カード）
- **ドリルダウン（L2）**: 月別推移チャート + 会議室別内訳
- **詳細データ（L3）**: 利用履歴テーブル

## 認知負荷への配慮

- ワーキングメモリの限界（4-5項目）を考慮: ダッシュボードのKPIカードは最大4枚
- Data-Ink Ratio を意識: グリッド線は主軸のみ。チャート背景は白（ライト）/ gray-900（ダーク）
- ゲシュタルトの法則に基づくグルーピング:
  - **近接**: 関連するKPIカードとチャートを近接配置
  - **類同**: 同じ集計区分のチャートは同系色で統一
  - **閉合**: Card コンポーネントでチャートをグループ化
- チャートのツールチップ: ホバー時に具体的な数値を表示。タッチデバイスはタップで表示
- 色の使用: 最大5色まで。それ以上は「その他」にまとめる
