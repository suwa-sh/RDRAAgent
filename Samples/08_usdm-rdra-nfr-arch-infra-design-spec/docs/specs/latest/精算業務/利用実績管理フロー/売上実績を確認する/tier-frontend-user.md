# 売上実績を確認する - 利用者・オーナー向けフロントエンド仕様

## 変更概要

会議室オーナーが自身の会議室の売上実績を確認するための画面を追加する。会議室別・期間別の集計軸を切り替えて売上金額・前期比を棒グラフ/折れ線グラフで可視化する。

## 画面仕様

### 売上実績確認画面

- **URL**: `/owner/sales`
- **アクセス権**: 会議室オーナー（ownerロール）
- **ポータル**: owner

#### UIロジック

- **状態管理**: 集計軸（dimension: 'room' | 'period'）と集計期間（from, to: YYYY-MM）をローカルステートで管理。集計条件変更時は即座に API を再フェッチする
- **バリデーション**: from <= to の期間整合性チェック。最大集計期間は12ヶ月。from/to が未入力の場合は集計実行ボタンを disabled にする
- **ローディング**: API フェッチ中はグラフ領域にスケルトンスクリーンを表示する
- **エラーハンドリング**: API エラー（500系）時は「データの取得に失敗しました。しばらくしてから再試行してください。」のトースト通知を表示する

#### 操作フロー

1. オーナーポータルサイドメニューから「売上実績」を選択する
2. デフォルト表示: 集計軸「物件別」・期間「直近3ヶ月（当月〜3ヶ月前）」で棒グラフを表示する
3. 集計軸タブを「期間別」に切り替えると折れ線グラフに切り替わる
4. 期間選択フィールドで開始月・終了月を変更し「集計」ボタンを押下する
5. 変更した条件でグラフとテーブルが再描画される

## コンポーネント設計

### SalesChartCard

- **ベースコンポーネント**: Card（default）
- **Props**:
  | Prop | 型 | 必須 | 説明 |
  |------|---|------|------|
  | dimension | `'room' \| 'period'` | Yes | 集計軸。'room'なら棒グラフ、'period'なら折れ線グラフ |
  | data | `SalesDataItem[]` | Yes | 集計データの配列 |
  | from | string | Yes | 集計開始年月（YYYY-MM） |
  | to | string | Yes | 集計終了年月（YYYY-MM） |
  | isLoading | boolean | No | ローディング状態 |
- **状態**: なし（純粋表示コンポーネント）
- **イベント**: なし

### SalesSummaryBanner

- **ベースコンポーネント**: Card（default）
- **Props**:
  | Prop | 型 | 必須 | 説明 |
  |------|---|------|------|
  | totalAmount | number | Yes | 集計期間の総売上金額（円） |
  | totalCount | number | Yes | 集計期間の総利用回数 |
  | growthRate | number | No | 前期比（%）。正値: 増加、負値: 減少 |
- **状態**: なし（純粋表示コンポーネント）
- **イベント**: なし
