## 凡例

### レベル定義

| Lv | 意味 |
|:--:|------|
| 0 | 規定なし・最低水準 |
| 1 | 低水準 |
| 2 | 標準水準 |
| 3 | 高水準 |
| 4 | 非常に高い水準 |
| 5 | 最高水準 |

### 確信度

| 確信度 | 意味 |
|:------:|------|
| 高 | RDRA モデルから明確に推論 |
| 中 | RDRA モデルから間接推論 |
| 低 | 弱い根拠での推論 |
| デフォルト | モデルシステムのデフォルト値 |
| ユーザー指定 | 対話でユーザーが指定 |
