## Tech Stack

- **Framework**: Next.js + TypeScript + Tailwind CSS v4
- **Design Tokens**: 3-layer (Primitive / Semantic / Component)
- **Dark Mode**: CSS class (`.dark`) + `prefers-color-scheme`
- **Portal Switching**: `data-portal` attribute on `<html>`

## Theme Switching

ツールバーの **Portal** と **Theme** セレクターで切り替えできます。
