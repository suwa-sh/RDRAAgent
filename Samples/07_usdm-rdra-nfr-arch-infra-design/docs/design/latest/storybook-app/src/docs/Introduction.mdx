# RoomConnect Design System

貸し会議室マッチングSaaS のデザインシステム。
RDRA/NFR/Arch モデルから導出した、3ポータル対応のコンポーネントライブラリ。

## Design Principles

1. **ポータル識別性** - 各ポータルはプライマリカラーで即座に識別可能
2. **信頼性** - 決済・審査フローには明確なステータス表示と確認ステップ
3. **レスポンシブ** - 利用者ポータルはモバイルファースト、管理系はデスクトップ最適化
4. **アクセシビリティ** - WCAG 2.1 AA 準拠、コントラスト比 4.5:1 以上
