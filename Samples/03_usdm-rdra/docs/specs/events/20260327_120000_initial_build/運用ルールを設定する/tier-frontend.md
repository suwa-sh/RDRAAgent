# 運用ルールを設定する - フロントエンド仕様

## 変更概要

会議室オーナーが会議室の運用ルールを設定・変更するための運用ルール設定画面を新規作成する。

## 画面仕様

### 運用ルール設定画面

- **URL**: /rooms/:room_id/operation-rules
- **アクセス権**: 会議室オーナー（自身の所有する会議室のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 対象会議室の名称を表示する |
| 利用可能時間帯（開始） | 時間入力 | 利用可能な開始時刻を入力する |
| 利用可能時間帯（終了） | 時間入力 | 利用可能な終了時刻を入力する |
| 最低利用時間 | 数値入力 | 最低利用時間を時間単位で入力する |
| 最大利用時間 | 数値入力 | 最大利用時間を時間単位で入力する |
| 貸出可否 | トグル | 貸出可・不可を切り替える |
| 保存ボタン | ボタン | 運用ルールを保存する |
| キャンセルボタン | ボタン | 変更を破棄して会議室詳細に戻る |

#### 操作フロー

1. 会議室オーナーが運用ルール設定画面を開く
2. 既存の運用ルールがある場合は現在の値がプリセットされる
3. 利用可能時間帯、最低利用時間、最大利用時間、貸出可否を入力・変更する
4. 保存ボタンをクリックする
5. バリデーションが成功した場合、運用ルールが保存され完了メッセージが表示される
6. バリデーションが失敗した場合、該当項目にエラーメッセージが表示される
