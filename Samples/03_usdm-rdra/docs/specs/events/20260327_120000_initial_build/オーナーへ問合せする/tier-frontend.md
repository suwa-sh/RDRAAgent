# オーナーへ問合せする - フロントエンド仕様

## 変更概要

オーナー問合せ画面を新規作成する。利用者が予約に関して会議室オーナーへ問合せを送信する。予約情報を参照しながら問合せ内容を入力できる。

## 画面仕様

### オーナー問合せ画面

- **URL**: /reservations/{reservation_id}/inquiries/new
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 予約情報 | テキスト | 予約に紐づく会議室名、利用日時を表示 |
| オーナー名 | テキスト | 問合せ先のオーナー名を表示 |
| 予約状態 | テキスト | 現在の予約状態を表示 |
| 問合せ内容 | テキストエリア | 問合せ内容を入力する |
| 送信ボタン | ボタン | 問合せを送信する |
| エラーメッセージ | テキスト | バリデーションエラー時に表示 |

#### 操作フロー

1. 利用者が予約詳細画面からオーナー問合せ画面に遷移する
2. 予約情報（会議室名、利用日時、予約状態）とオーナー名が表示される
3. 問合せ内容を入力する
4. 送信ボタンを押下する
5. 問合せ送信完了メッセージが表示される
