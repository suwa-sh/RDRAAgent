# オーナーへ問合せする

## 概要

利用者が予約に関して会議室オーナーへ問合せを行う。会議室予約フロー内での問合せであり、予約に紐づく形でオーナーへ問合せ情報を作成する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 会議室予約フロー | このUCを含むBUC |
| アクター | 利用者 | 操作するアクター |
| 情報 | 問合せ | 作成する問合せ情報 |
| バリエーション | 問合せ種別 | オーナー宛問合せ |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: オーナーへ問合せする

  Scenario: 予約に関してオーナーへ問合せする
    Given 利用者「田中太郎」がログイン済み
    And 予約「R001」が存在し会議室オーナー「佐藤花子」に紐づいている
    When 予約「R001」に関して問合せ内容「利用当日の入館方法を教えてください」を送信する
    Then 問合せが作成される
    And 問合せ先区分が「オーナー宛問合せ」である
    And 問合せ送信完了メッセージが表示される

  Scenario: 予約の詳細を確認しながら問合せする
    Given 利用者「田中太郎」がログイン済み
    And 予約「R001」が「確定」状態で存在する（会議室: 新宿カンファレンスルームA、利用日時: 2026-04-10 10:00〜12:00）
    When オーナー問合せ画面を開く
    Then 予約情報（会議室名、利用日時）が表示される
    And 問合せ内容入力欄が表示される
```

### 異常系

```gherkin
  Scenario: 問合せ内容が空で送信しようとする
    Given 利用者「田中太郎」がログイン済み
    When 問合せ内容を空のまま送信しようとする
    Then 「問合せ内容を入力してください」とエラーが表示される
    And 問合せは作成されない
```
