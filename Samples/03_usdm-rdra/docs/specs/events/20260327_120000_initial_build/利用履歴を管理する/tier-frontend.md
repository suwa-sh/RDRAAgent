# 利用履歴を管理する - フロントエンド仕様

## 変更概要

利用履歴管理画面を新規作成する。サービス運営担当者が利用履歴の一覧表示・検索・詳細確認を行えるようにする。

## 画面仕様

### 利用履歴管理画面

- **URL**: /admin/usage-history
- **アクセス権**: サービス運営担当者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 利用者名検索 | テキスト入力 | 利用者名で利用履歴を絞り込む |
| 会議室名検索 | テキスト入力 | 会議室名で利用履歴を絞り込む |
| 対象期間（開始） | 日付入力 | 検索対象期間の開始日 |
| 対象期間（終了） | 日付入力 | 検索対象期間の終了日 |
| 検索ボタン | ボタン | 条件に基づき利用履歴を検索 |
| 利用履歴一覧 | テーブル | 履歴ID、利用者名、会議室名、利用日時、利用時間、利用料金 |
| ページネーション | ナビゲーション | 一覧のページ切り替え |
| 詳細リンク | リンク | 各行の詳細画面への遷移 |

#### 操作フロー

1. サービス運営担当者が利用履歴管理画面にアクセスする
2. 必要に応じて利用者名・会議室名・対象期間で検索条件を指定する
3. 検索ボタンを押下する
4. 条件に合致する利用履歴が一覧表示される
5. 詳細を確認したい行のリンクをクリックし、詳細情報を確認する
