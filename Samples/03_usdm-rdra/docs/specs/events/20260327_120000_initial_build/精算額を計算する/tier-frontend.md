# 精算額を計算する - フロントエンド仕様

## 変更概要

精算額計算画面を新規作成する。サービス運営担当者が対象月を指定してオーナーへの精算額を計算・確認できるようにする。

## 画面仕様

### 精算額計算画面

- **URL**: /admin/settlement/calculate
- **アクセス権**: サービス運営担当者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 精算対象月 | 月選択 | 精算対象の年月を選択（YYYY-MM） |
| 計算ボタン | ボタン | 精算額の計算を実行 |
| 精算一覧 | テーブル | オーナー名、利用料合計、手数料合計、精算額、精算状態 |
| 内訳リンク | リンク | 各オーナーの精算内訳詳細への遷移 |
| 精算内訳パネル | パネル | 精算ID、精算対象月、利用料合計、手数料合計、精算額、精算状態、精算日 |

#### 操作フロー

1. サービス運営担当者が精算額計算画面にアクセスする
2. 精算対象月を選択する
3. 計算ボタンを押下する
4. 対象月の全オーナーの精算額が一覧に表示される
5. 各オーナーの内訳リンクをクリックして精算内訳を確認する

## ティア完了条件（BDD）

```gherkin

  Scenario: 精算対象月を指定して計算する
    Given 精算額計算画面が表示されている

    Given 精算額計算画面に精算一覧が表示されている

    Then 精算内訳パネルに精算ID・利用料合計・手数料合計・精算額が表示される

  Scenario: 既に精算計算済みの月を選択した場合
    Given 精算額計算画面が表示されている

    When 精算対象月を「2026-03」に設定し計算ボタンを押下する
    Then 「既に精算計算済みです。再計算する場合は確認してください」と警告が表示される
```
