# 利用状況を分析する

## 概要

サービス運営担当者が会議室利用状況を集計・分析し、利用傾向を把握する。利用履歴と売上実績をもとに、会員別・物件別・期間別の多角的な分析を提供する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 利用状況管理フロー | このUCを含むBUC |
| アクター | サービス運営担当者 | 操作するアクター |
| 情報 | 利用履歴 | 参照する情報（会員別・物件別の利用履歴） |
| 情報 | 売上実績 | 参照する情報（会議室ごとの利用回数・売上金額） |
| バリエーション | 利用履歴集計区分 | 会員別、物件別、期間別 |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 利用状況を分析する

  Scenario: 会員別の利用状況を分析する
    Given サービス運営担当者「山田花子」がログイン済み
    And 利用者「田中太郎」の2026年3月の利用履歴が5件存在する
    When 利用履歴集計区分「会員別」で2026年3月の利用状況分析画面を表示する
    Then 利用者「田中太郎」の利用回数5回・利用料金合計が表示される

  Scenario: 物件別の利用状況を分析する
    Given サービス運営担当者「山田花子」がログイン済み
    And 会議室「新宿カンファレンスルームA」の2026年3月の利用履歴が8件存在する
    When 利用履歴集計区分「物件別」で2026年3月の利用状況分析画面を表示する
    Then 会議室「新宿カンファレンスルームA」の利用回数8回・売上金額合計が表示される

  Scenario: 期間別の利用状況推移を確認する
    Given サービス運営担当者「山田花子」がログイン済み
    And 2026年1月から3月の利用履歴データが存在する
    When 利用履歴集計区分「期間別」で2026年1月から3月の利用状況分析画面を表示する
    Then 月ごとの利用回数・売上金額の推移が表示される
```

### 異常系

```gherkin
  Scenario: 対象期間にデータが存在しない場合
    Given サービス運営担当者「山田花子」がログイン済み
    And 2025年1月の利用履歴データが存在しない
    When 利用履歴集計区分「期間別」で2025年1月の利用状況分析画面を表示する
    Then 「該当期間の利用状況データがありません」とメッセージが表示される

  Scenario: サービス運営担当者以外がアクセスした場合
    Given 利用者「佐藤次郎」がログイン済み
    When 利用状況分析画面にアクセスする
    Then 「アクセス権限がありません」とエラーが表示される
```
