# 利用回数を確認する - フロントエンド仕様

## 変更概要

利用回数確認画面を新規作成する。会議室オーナーが自身の会議室の利用回数実績を確認できるようにする。

## 画面仕様

### 利用回数確認画面

- **URL**: /owner/usage-count
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 対象期間（開始） | 日付入力 | 確認対象期間の開始日 |
| 対象期間（終了） | 日付入力 | 確認対象期間の終了日 |
| 検索ボタン | ボタン | 条件に基づき利用回数を検索 |
| 会議室別利用回数一覧 | テーブル | 会議室名、利用回数、利用時間合計 |
| 利用履歴詳細 | テーブル | 利用日時、利用者名、利用時間、利用料金 |

#### 操作フロー

1. 会議室オーナーが利用回数確認画面にアクセスする
2. 対象期間を指定する（デフォルトは当月）
3. 検索ボタンを押下する
4. 自身の会議室ごとの利用回数が一覧で表示される
5. 会議室を選択すると利用履歴の詳細が表示される
