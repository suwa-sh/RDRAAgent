# オーナー登録審査一覧を確認する - フロントエンド仕様

## 変更概要

サービス運営担当者がオーナー登録申請の審査状況を一覧形式で確認・管理するオーナー審査管理画面を新規作成する。

## 画面仕様

### オーナー審査管理画面

- **URL**: /admin/owners/reviews
- **アクセス権**: サービス運営担当者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 審査状態フィルタ | セレクトボックス | 「全て」「審査待ち」「登録済み」「却下」「退会」で一覧をフィルタする |
| オーナー一覧テーブル | テーブル | オーナー申請情報の一覧を表示する |
| オーナーID列 | テキスト | オーナーの一意識別子 |
| 氏名列 | テキスト | 申請者の氏名 |
| メールアドレス列 | テキスト | 申請者のメールアドレス |
| 登録日列 | テキスト | 申請日 |
| 審査状態列 | テキスト | 現在の審査状態 |
| 詳細リンク | リンク | 個別のオーナー審査画面への遷移リンク |
| 空一覧メッセージ | テキスト | 該当するオーナーが0件の場合に表示するメッセージ |

#### 操作フロー

1. サービス運営担当者がオーナー審査管理画面にアクセスする
2. デフォルトで「審査待ち」のオーナー一覧が表示される
3. 必要に応じて審査状態フィルタで表示対象を切り替える
4. 一覧の行またはリンクをクリックして個別審査画面に遷移する

    When オーナー「鈴木一郎」の行をクリックする
