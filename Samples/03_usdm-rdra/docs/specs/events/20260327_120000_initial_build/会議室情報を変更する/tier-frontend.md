# 会議室情報を変更する - フロントエンド仕様

## 変更概要

会議室オーナーが登録済みの会議室情報を修正・更新するための会議室情報編集画面を新規作成する。

## 画面仕様

### 会議室情報編集画面

- **URL**: /rooms/:room_id/edit
- **アクセス権**: 会議室オーナー（自身の所有する会議室のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト入力 | 必須。現在の会議室名がプリセットされる |
| 所在地 | テキスト入力 | 必須。現在の所在地がプリセットされる |
| 広さ | 数値入力 | 必須。現在の値がプリセットされる |
| 収容人数 | 数値入力 | 必須。現在の値がプリセットされる |
| 価格 | 数値入力 | 必須。現在の値がプリセットされる |
| 設備・機能 | チェックボックス | 現在の選択状態がプリセットされる |
| 画像 | ファイルアップロード | 既存画像の表示・削除・追加が可能 |
| 保存ボタン | ボタン | 変更内容を保存する |
| キャンセルボタン | ボタン | 変更を破棄して詳細画面に戻る |

#### 操作フロー

1. 会議室オーナーが会議室一覧から対象の会議室を選択する
2. 会議室情報編集画面が開き、現在の情報がフォームにプリセットされる
3. 変更したい項目を修正する
4. 保存ボタンをクリックする
5. バリデーションが成功した場合、情報が更新され完了メッセージが表示される
6. バリデーションが失敗した場合、該当項目にエラーメッセージが表示される
