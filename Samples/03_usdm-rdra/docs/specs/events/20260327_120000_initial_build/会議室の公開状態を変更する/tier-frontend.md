# 会議室の公開状態を変更する - フロントエンド仕様

## 変更概要

会議室オーナーが会議室の公開状態を切り替えるための会議室公開管理画面を新規作成する。

## 画面仕様

### 会議室公開管理画面

- **URL**: /rooms/:room_id/publication
- **アクセス権**: 会議室オーナー（自身の所有する会議室のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 対象会議室の名称を表示する |
| 現在の公開状態 | テキスト | 現在の公開状態（非公開/公開可能/公開中）を表示する |
| 公開条件チェックリスト | テーブル | 会議室公開条件の充足状況を一覧表示する |
| 公開状態変更ボタン | ボタン | 遷移可能な状態への変更ボタンを表示する |

#### 操作フロー

1. 会議室オーナーが会議室公開管理画面を開く
2. 現在の公開状態と公開条件の充足状況が表示される
3. 遷移可能な公開状態のボタンをクリックする
4. 公開条件を満たしている場合、確認ダイアログが表示される
5. 確認後、公開状態が変更され完了メッセージが表示される
6. 公開条件を満たしていない場合、未充足の条件がエラーとして表示される
