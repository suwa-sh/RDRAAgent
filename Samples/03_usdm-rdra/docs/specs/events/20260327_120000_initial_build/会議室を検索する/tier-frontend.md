# 会議室を検索する - フロントエンド仕様

## 変更概要

会議室検索画面を新規作成する。利用者が複数の検索条件を指定して公開中の会議室を絞り込み、検索結果を一覧表示する。

## 画面仕様

### 会議室検索画面

- **URL**: /rooms/search
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| エリア | セレクトボックス | 所在地エリアの選択 |
| 広さ | 数値入力 | 最小広さ（平米） |
| 収容人数 | 数値入力 | 最小収容人数 |
| 価格帯（下限） | 数値入力 | 1時間あたりの最低価格 |
| 価格帯（上限） | 数値入力 | 1時間あたりの最高価格 |
| 設備・機能 | チェックボックス群 | プロジェクター、ホワイトボード、Wi-Fi、テレビ会議設備、音響設備、空調、飲食可、喫煙可 |
| 評価スコア | セレクトボックス | 最低評価スコア |
| 利用可能日時 | 日時ピッカー | 利用希望日時 |
| 検索ボタン | ボタン | 検索を実行する |
| 検索結果一覧 | テーブル | 会議室名、所在地、広さ、収容人数、価格、評価スコア、設備アイコンを表示 |
| 検索結果0件メッセージ | テキスト | 条件に一致する会議室がない場合に表示 |

#### 操作フロー

1. 利用者が会議室検索画面にアクセスする
2. 検索条件（エリア、広さ、収容人数、価格帯、設備・機能、評価スコア、利用可能日時）を入力する
3. 検索ボタンを押下する
4. 条件に一致する公開中の会議室が一覧表示される
5. 一覧から会議室を選択すると会議室詳細画面へ遷移する
