# 問合せに対応する - フロントエンド仕様

## 変更概要

問合せ対応画面を新規作成する。サービス運営担当者が利用者からの問合せ一覧を確認し、回答済みの問合せを対応済みに変更できるようにする。

## 画面仕様

### 問合せ対応画面

- **URL**: /admin/inquiries
- **アクセス権**: サービス運営担当者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 問合せ状態フィルタ | セレクトボックス | 全て、未対応、回答済み、対応済みで絞り込み |
| 問合せ一覧 | テーブル | 問合せID、利用者名、問合せ内容（先頭50文字）、問合せ状態、問合せ日時 |
| 問合せ詳細パネル | パネル | 問合せの全内容・回答内容・状態遷移ボタン |
| 対応済みボタン | ボタン | 回答済みの問合せを対応済みに変更 |

#### 操作フロー

1. サービス運営担当者が問合せ対応画面にアクセスする
2. 問合せ一覧から対応する問合せを選択する
3. 問合せ詳細パネルで問合せ内容と回答内容を確認する
4. 回答済みの問合せの場合、対応済みボタンを押下して対応完了とする
