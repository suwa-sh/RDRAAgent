# 決済方法を設定する - フロントエンド仕様

## 変更概要

決済方法設定画面を新規作成する。利用者がクレジットカードまたは電子マネーの決済手段を選択し、必要な情報を入力して登録する。

## 画面仕様

### 決済方法設定画面

- **URL**: /reservations/{reservation_id}/payment
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 決済方法選択 | ラジオボタン | クレジットカード、電子マネーから選択 |
| カード番号 | テキスト入力 | クレジットカード番号（決済方法がクレジットカードの場合） |
| 有効期限 | 月年ピッカー | カード有効期限（決済方法がクレジットカードの場合） |
| セキュリティコード | テキスト入力 | カードのCVV/CVC（決済方法がクレジットカードの場合） |
| 電子マネーアカウント | テキスト入力 | 電子マネーアカウント情報（決済方法が電子マネーの場合） |
| 登録ボタン | ボタン | 決済方法を登録する |
| エラーメッセージ | テキスト | バリデーションエラー時に表示 |

#### 操作フロー

1. 利用者が決済方法設定画面にアクセスする
2. 決済方法（クレジットカードまたは電子マネー）を選択する
3. 選択した決済方法に応じた入力項目が表示される
4. 必要な情報を入力する
5. 登録ボタンを押下する
6. 決済方法設定完了メッセージが表示される
