# 手数料売上を分析する - フロントエンド仕様

## 変更概要

手数料売上分析画面を新規作成する。サービス運営担当者が売上分析区分（会議室別、貸出別、月別、オーナー別）を切り替えて手数料売上データを集計・表示できるようにする。

## 画面仕様

### 手数料売上分析画面

- **URL**: /admin/commission-analysis
- **アクセス権**: サービス運営担当者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 売上分析区分セレクタ | セレクトボックス | 会議室別、貸出別、月別、オーナー別を選択 |
| 対象期間（開始） | 日付入力 | 分析対象期間の開始日 |
| 対象期間（終了） | 日付入力 | 分析対象期間の終了日 |
| 検索ボタン | ボタン | 条件に基づき手数料売上を検索・集計 |
| 手数料売上一覧 | テーブル | 手数料ID、会議室ID、貸出ID、手数料率、手数料金額、計上日 |
| 売上合計 | テキスト | 表示中データの手数料金額合計 |
| CSVエクスポートボタン | ボタン | 表示中の分析結果をCSVでダウンロード |

#### 操作フロー

1. サービス運営担当者が手数料売上分析画面にアクセスする
2. 売上分析区分（会議室別/貸出別/月別/オーナー別）を選択する
3. 対象期間を指定する
4. 検索ボタンを押下する
5. 選択した分析区分に応じた集計結果がテーブルに表示される
6. 必要に応じてCSVエクスポートボタンでデータをダウンロードする
