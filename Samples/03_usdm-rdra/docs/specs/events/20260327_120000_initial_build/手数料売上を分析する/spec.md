# 手数料売上を分析する

## 概要

サービス運営担当者が手数料売上を会議室別・貸出別に集計・分析し、サービス運営の収益状況を把握する。売上分析区分（会議室別、貸出別、月別、オーナー別）に応じた多角的な分析を提供する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | サービス運営管理フロー | このUCを含むBUC |
| アクター | サービス運営担当者 | 操作するアクター |
| 情報 | 手数料売上 | 参照する情報（会議室別・貸出別の手数料売上データ） |
| 情報 | 利用履歴 | 参照する情報（利用実績の基礎データ） |
| バリエーション | 売上分析区分 | 会議室別、貸出別、月別、オーナー別 |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 手数料売上を分析する

  Scenario: 会議室別の手数料売上を集計する
    Given サービス運営担当者「山田花子」がログイン済み
    And 会議室「新宿カンファレンスルームA」の2026年3月の手数料売上が5件登録されている
    When 売上分析区分「会議室別」で2026年3月の手数料売上分析画面を表示する
    Then 会議室「新宿カンファレンスルームA」の手数料売上合計が表示される
    And 会議室ごとの手数料金額・手数料率・計上日が一覧で確認できる

  Scenario: 貸出別の手数料売上を集計する
    Given サービス運営担当者「山田花子」がログイン済み
    And 2026年3月の貸出に対する手数料売上が10件登録されている
    When 売上分析区分「貸出別」で2026年3月の手数料売上分析画面を表示する
    Then 貸出ごとの手数料ID・会議室名・手数料金額・計上日が一覧で表示される

  Scenario: 月別の手数料売上推移を確認する
    Given サービス運営担当者「山田花子」がログイン済み
    And 2026年1月から3月までの手数料売上データが存在する
    When 売上分析区分「月別」で2026年1月から3月の手数料売上分析画面を表示する
    Then 月ごとの手数料売上合計が時系列で表示される

  Scenario: オーナー別の手数料売上を集計する
    Given サービス運営担当者「山田花子」がログイン済み
    And オーナー「鈴木一郎」に紐づく会議室の手数料売上が存在する
    When 売上分析区分「オーナー別」で手数料売上分析画面を表示する
    Then オーナー「鈴木一郎」の手数料売上合計が表示される
```

### 異常系

```gherkin
  Scenario: 対象期間にデータが存在しない場合
    Given サービス運営担当者「山田花子」がログイン済み
    And 2025年1月の手数料売上データが存在しない
    When 売上分析区分「月別」で2025年1月の手数料売上分析画面を表示する
    Then 「該当期間の手数料売上データがありません」とメッセージが表示される

  Scenario: サービス運営担当者以外がアクセスした場合
    Given 利用者「佐藤次郎」がログイン済み
    When 手数料売上分析画面にアクセスする
    Then 「アクセス権限がありません」とエラーが表示される
```
