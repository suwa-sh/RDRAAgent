# 予約を取り消す - フロントエンド仕様

## 変更概要

予約取消画面を新規作成する。利用者が予約の取消を行い、キャンセルポリシーに基づくキャンセル料を確認した上で取消を実行する。

## 画面仕様

### 予約取消画面

- **URL**: /reservations/{reservation_id}/cancel
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 予約対象の会議室名を表示 |
| 予約日時 | テキスト | 利用開始・終了日時を表示 |
| 予約状態 | テキスト | 現在の予約状態を表示 |
| キャンセルポリシー説明 | テキスト | 適用されるキャンセルポリシーの内容を表示 |
| キャンセル料 | テキスト | 計算されたキャンセル料を表示 |
| 取消確認メッセージ | テキスト | 「本当に取り消しますか？」の確認メッセージ |
| 取消実行ボタン | ボタン | 予約取消を実行する |
| キャンセルボタン | ボタン | 取消を取りやめ予約詳細に戻る |

#### 操作フロー

1. 利用者が予約詳細画面から予約取消画面に遷移する
2. 予約内容とキャンセルポリシー、計算されたキャンセル料が表示される
3. 取消確認メッセージを確認する
4. 取消実行ボタンを押下する
5. 予約取消完了メッセージが表示される
