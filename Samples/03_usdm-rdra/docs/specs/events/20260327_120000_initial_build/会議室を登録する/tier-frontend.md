# 会議室を登録する - フロントエンド仕様

## 変更概要

会議室オーナーが会議室の物件情報を入力・登録するための会議室登録画面を新規作成する。

## 画面仕様

### 会議室登録画面

- **URL**: /rooms/new
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト入力 | 必須。会議室の名称を入力する |
| 所在地 | テキスト入力 | 必須。会議室の住所を入力する |
| 広さ | 数値入力 | 必須。平方メートル単位で入力する |
| 収容人数 | 数値入力 | 必須。1以上の整数を入力する |
| 価格 | 数値入力 | 必須。時間あたりの利用料金を入力する |
| 設備・機能 | チェックボックス | プロジェクター、ホワイトボード、Wi-Fi、テレビ会議設備、音響設備、空調、飲食可、喫煙可から選択 |
| 画像 | ファイルアップロード | 会議室の写真を複数枚アップロード可能 |
| 登録ボタン | ボタン | 入力内容を確認し会議室を登録する |
| キャンセルボタン | ボタン | 入力内容を破棄して一覧に戻る |

#### 操作フロー

1. 会議室オーナーが会議室登録画面を開く
2. 会議室名、所在地、広さ、収容人数、価格を入力する
3. 設備・機能をチェックボックスから選択する
4. 必要に応じて会議室の画像をアップロードする
5. 登録ボタンをクリックする
6. バリデーションが成功した場合、会議室が登録され完了メッセージが表示される
7. バリデーションが失敗した場合、該当項目にエラーメッセージが表示される
