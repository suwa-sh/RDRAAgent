# 利用者評価一覧を確認する - フロントエンド仕様

## 変更概要

会議室オーナーが利用者から受けた評価を一覧で確認する利用者評価一覧画面を新規作成する。

## 画面仕様

### 利用者評価一覧画面

- **URL**: /owner/ratings/received
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 評価サマリ | テキスト | 全評価の平均スコア、総評価件数 |
| 評価一覧 | テーブル | 利用者名、会議室名、評価スコア、コメント、評価日時 |
| 評価なしメッセージ | テキスト | 評価が0件の場合に表示されるメッセージ |
| ソート切替 | ボタン | 評価日時順/評価スコア順の切替 |

#### 操作フロー

1. 会議室オーナーが利用者評価一覧画面を開く
2. 評価サマリ（平均スコア、総件数）が表示される
3. 評価一覧が新しい順に表示される
4. 必要に応じてソート順を切り替える
5. 各評価の詳細を確認する
