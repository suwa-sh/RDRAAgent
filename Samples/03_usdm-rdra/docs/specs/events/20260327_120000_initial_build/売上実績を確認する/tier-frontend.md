# 売上実績を確認する - フロントエンド仕様

## 変更概要

売上実績確認画面を新規作成する。会議室オーナーが自身の会議室の売上実績（利用回数・売上金額）を確認できるようにする。

## 画面仕様

### 売上実績確認画面

- **URL**: /owner/sales
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 対象期間（開始） | 日付入力 | 確認対象期間の開始日 |
| 対象期間（終了） | 日付入力 | 確認対象期間の終了日 |
| 検索ボタン | ボタン | 条件に基づき売上実績を検索 |
| 売上実績一覧 | テーブル | 会議室名、集計期間、利用回数、売上金額 |
| 売上合計 | テキスト | 表示中の全会議室の売上金額合計 |

#### 操作フロー

1. 会議室オーナーが売上実績確認画面にアクセスする
2. 対象期間を指定する（デフォルトは当月）
3. 検索ボタンを押下する
4. 自身の会議室ごとの売上実績が一覧で表示される
5. 売上合計で全体の収益を確認する
