# 予約を申請する - フロントエンド仕様

## 変更概要

予約申請画面を新規作成する。利用者が会議室の利用日時を指定し予約申請を送信する。

## 画面仕様

### 予約申請画面

- **URL**: /rooms/{room_id}/reservations/new
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 予約対象の会議室名を表示 |
| 会議室画像 | 画像 | 会議室の画像を表示 |
| 利用日 | 日付ピッカー | 利用希望日を選択 |
| 利用開始時間 | 時刻ピッカー | 利用開始時間を選択 |
| 利用終了時間 | 時刻ピッカー | 利用終了時間を選択 |
| 利用料金見積 | テキスト | 選択した時間に基づく利用料金を表示 |
| 予約申請ボタン | ボタン | 予約申請を送信する |
| エラーメッセージ | テキスト | バリデーションエラー時に表示 |

#### 操作フロー

1. 利用者が会議室詳細画面から予約申請画面に遷移する
2. 利用日、利用開始時間、利用終了時間を選択する
3. 利用料金見積が自動計算・表示される
4. 予約申請ボタンを押下する
5. 予約申請完了メッセージが表示される
