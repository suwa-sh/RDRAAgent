# バーチャル会議室利用を終了する - バックエンド仕様

## 変更概要

バーチャル会議室の利用終了時刻到来時に自動的に利用終了状態へ遷移するバッチ処理を新規作成する。

## API 仕様

### バーチャル会議室利用終了バッチ

- **種別**: スケジュールバッチ（毎分実行）
- **トリガー**: 利用終了時刻到来（タイマー）

#### 処理対象

| 条件 | 説明 |
|------|------|
| 会議室種別 | バーチャル |
| 会議室利用状態 | 利用中 |
| 利用終了日時 | 現在日時以前 |

#### 処理内容

1. 条件に一致する会議室利用を検索する
2. 利用状態を「利用終了」に更新する
3. 利用終了日時を記録する

## データモデル変更

なし（既存の会議室利用テーブルを使用）

## ビジネスルール

- バーチャル会議室利用ポリシーに基づき、鍵の返却は不要
- 会議室種別が「バーチャル」かつ利用状態が「利用中」の利用のみが対象
- 利用終了時刻到来で自動的に利用終了状態に遷移する
- 物理会議室は本バッチの対象外（従来の鍵返却フローを適用）
