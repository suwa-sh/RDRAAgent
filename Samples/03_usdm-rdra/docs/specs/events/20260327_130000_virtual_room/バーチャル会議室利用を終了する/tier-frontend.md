# バーチャル会議室利用を終了する - フロントエンド仕様

## 変更概要

バーチャル会議室の利用終了はタイマーによる自動処理のため、専用の操作画面は不要。利用者の予約一覧画面で利用状態の表示を更新する。

## 画面仕様

### 予約一覧画面（既存画面の変更）

- **URL**: /reservations
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 利用状態 | テキスト | バーチャル会議室の場合、利用終了時刻到来で「利用終了」に自動更新 |
| 会議URLリンク | リンク | 利用終了後は会議URLリンクを非表示にする |

#### 操作フロー

1. 利用者が予約一覧画面を開く
2. バーチャル会議室の予約について、利用終了時刻到来後に利用状態が「利用終了」と表示される
3. 利用終了後は会議URLリンクが非表示になる
