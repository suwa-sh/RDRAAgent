# 会議URLを通知する - フロントエンド仕様

## 変更概要

会議URL通知画面を新規作成する。バーチャル会議室の予約確定後に利用者が会議URL情報を確認できる画面を提供する。

## 画面仕様

### 会議URL通知画面

- **URL**: /reservations/{reservation_id}/meeting-url
- **アクセス権**: 利用者（予約者本人のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 予約したバーチャル会議室の名称 |
| 会議ツール種別 | テキスト | Zoom/Teams/Google Meet の表示 |
| 会議URL | リンク | クリック可能な会議URLリンク |
| 予約日時 | テキスト | 利用開始日時〜利用終了日時 |
| 有効期限 | テキスト | 会議URLの有効期限 |
| URLコピーボタン | ボタン | 会議URLをクリップボードにコピーする |

#### 操作フロー

1. 利用者が予約確定の通知メールを受信する
2. 通知メール内のリンクまたは予約一覧から会議URL通知画面にアクセスする
3. 会議URL情報（会議室名・会議ツール種別・会議URL・予約日時）を確認する
4. 会議URLリンクをクリックして会議ツールに接続する、またはURLコピーボタンでURLをコピーする
