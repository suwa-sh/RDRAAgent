# バーチャル会議室オーナーを評価する - フロントエンド仕様

## 変更概要

オーナー評価登録画面をバーチャル会議室オーナーの評価にも対応させる。既存のオーナー評価登録画面を共用し、バーチャル会議室利用後のオーナー評価を登録できるようにする。

## 画面仕様

### オーナー評価登録画面（既存画面の変更）

- **URL**: /rooms/{room_id}/owner-reviews/new
- **アクセス権**: 利用者（利用済みの場合のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 利用したバーチャル会議室名 |
| 会議室種別 | テキスト | 「バーチャル」と表示 |
| オーナー名 | テキスト | 評価対象のオーナー名 |
| 評価スコア | セレクトボックス | 1〜5の評価スコア（必須） |
| コメント | テキストエリア | 評価コメント（任意） |
| 評価登録ボタン | ボタン | 評価を登録する |

#### 操作フロー

1. 利用者が利用済みバーチャル会議室のオーナー評価登録画面にアクセスする
2. 会議室名・会議室種別・オーナー名が表示される
3. 評価スコアを選択し、コメントを入力する
4. 評価登録ボタンを押下する
5. 評価登録完了後、評価一覧画面へ遷移する
