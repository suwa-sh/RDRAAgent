# 会議室を評価する - フロントエンド仕様

## 変更概要

会議室評価登録画面を新規作成する。利用者が利用済みの会議室に対して評価スコアとコメントを登録する。

## 画面仕様

### 会議室評価登録画面

- **URL**: /rooms/{room_id}/reviews/new
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 評価対象の会議室名を表示 |
| 会議室画像 | 画像 | 会議室の画像を表示 |
| 利用日時 | テキスト | 利用した日時を表示 |
| 評価スコア | 星評価 | 1〜5の星評価を選択 |
| コメント | テキストエリア | 評価コメントを入力（任意） |
| 登録ボタン | ボタン | 評価を登録する |
| エラーメッセージ | テキスト | バリデーションエラー時に表示 |

#### 操作フロー

1. 利用者が利用履歴画面から会議室評価登録画面に遷移する
2. 会議室名と利用日時が表示される
3. 評価スコア（1〜5）を選択する
4. コメントを入力する（任意）
5. 登録ボタンを押下する
6. 評価登録完了メッセージが表示される
