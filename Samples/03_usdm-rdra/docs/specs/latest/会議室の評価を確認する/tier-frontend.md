# 会議室の評価を確認する - フロントエンド仕様

## 変更概要

会議室オーナーが利用者から付けられた評価を一覧で確認するための会議室評価一覧画面を新規作成する。

## 画面仕様

### 会議室評価一覧画面

- **URL**: /rooms/:room_id/reviews
- **アクセス権**: 会議室オーナー（自身の所有する会議室のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 対象会議室の名称を表示する |
| 平均評価スコア | テキスト | 全評価の平均スコアを表示する |
| 評価件数 | テキスト | 評価の総件数を表示する |
| 評価一覧 | テーブル | 評価スコア、コメント、評価者名、評価日時を一覧表示する |
| 評価なしメッセージ | テキスト | 評価がない場合に「まだ評価がありません」を表示する |

#### 操作フロー

1. 会議室オーナーが会議室評価一覧画面を開く
2. 会議室名と平均評価スコア、評価件数が表示される
3. 評価一覧が評価日時の新しい順に表示される
4. 評価がない場合は「まだ評価がありません」と表示される
