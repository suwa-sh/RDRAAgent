# 会議室の評価を確認する

## 概要

会議室オーナーが利用者から付けられた会議室の評価（評価スコア、コメント）を一覧で確認するユースケース。オーナーは評価内容を把握し、会議室運営の改善に活用する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 会議室管理フロー | このUCを含むBUC |
| アクター | 会議室オーナー | 操作するアクター |
| 情報 | 会議室評価 | 確認対象の情報 |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 会議室の評価を確認する

  Scenario: 会議室オーナーが評価一覧を確認する
    Given 会議室オーナー「山田太郎」がログイン済み
    And 会議室「渋谷ミーティングルームA」に利用者「鈴木一郎」から評価スコア「4」コメント「設備が充実していて快適でした」の評価がある
    When 会議室評価一覧画面を開く
    Then 利用者「鈴木一郎」の評価スコア「4」とコメント「設備が充実していて快適でした」が表示される

  Scenario: 複数の評価を一覧で確認する
    Given 会議室オーナー「山田太郎」がログイン済み
    And 会議室「渋谷ミーティングルームA」に3件の評価がある
    When 会議室評価一覧画面を開く
    Then 3件の評価が評価日時の新しい順に表示される

  Scenario: 評価がない場合の表示
    Given 会議室オーナー「山田太郎」がログイン済み
    And 会議室「渋谷ミーティングルームA」にまだ評価がない
    When 会議室評価一覧画面を開く
    Then 「まだ評価がありません」と表示される
```

### 異常系

```gherkin
  Scenario: 他のオーナーの会議室評価は確認できない
    Given 会議室オーナー「山田太郎」がログイン済み
    And 会議室「新宿会議室B」はオーナー「佐藤花子」の所有
    When 会議室「新宿会議室B」の評価一覧画面にアクセスする
    Then 「この会議室の評価を閲覧する権限がありません」とエラーが表示される
```
