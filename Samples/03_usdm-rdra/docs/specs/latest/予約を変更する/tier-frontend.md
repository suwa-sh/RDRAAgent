# 予約を変更する - フロントエンド仕様

## 変更概要

予約変更画面を新規作成する。利用者が確定済みの予約の利用日時を変更する。

## 画面仕様

### 予約変更画面

- **URL**: /reservations/{reservation_id}/edit
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 予約対象の会議室名を表示 |
| 現在の予約日時 | テキスト | 現在の利用開始・終了日時を表示 |
| 新しい利用日 | 日付ピッカー | 変更後の利用希望日を選択 |
| 新しい利用開始時間 | 時刻ピッカー | 変更後の利用開始時間を選択 |
| 新しい利用終了時間 | 時刻ピッカー | 変更後の利用終了時間を選択 |
| 変更後の利用料金見積 | テキスト | 変更後の時間に基づく利用料金を表示 |
| 変更ボタン | ボタン | 予約変更を送信する |
| キャンセルボタン | ボタン | 変更を取りやめ予約詳細に戻る |
| エラーメッセージ | テキスト | バリデーションエラー時に表示 |

#### 操作フロー

1. 利用者が予約詳細画面から予約変更画面に遷移する
2. 現在の予約日時が表示される
3. 新しい利用日、利用開始時間、利用終了時間を選択する
4. 変更後の利用料金見積が自動計算・表示される
5. 変更ボタンを押下する
6. 予約変更完了メッセージが表示される
