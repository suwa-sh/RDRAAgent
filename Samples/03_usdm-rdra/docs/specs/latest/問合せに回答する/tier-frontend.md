# 問合せに回答する - フロントエンド仕様

## 変更概要

会議室オーナーが利用者からの問合せ一覧を確認し、回答を入力する問合せ回答画面を新規作成する。

## 画面仕様

### 問合せ回答画面

- **URL**: /owner/inquiries/reply
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 未対応問合せ一覧 | テーブル | 未対応の問合せ一覧（問合せID、利用者名、問合せ内容抜粋、問合せ日時） |
| 問合せ詳細 | テキスト | 選択した問合せの全文（問合せ内容、利用者名、問合せ日時） |
| 回答入力欄 | フォーム | 回答内容を入力するテキストエリア |
| 回答ボタン | ボタン | 回答を送信する |
| 確認ダイアログ | テキスト | 回答送信前の確認メッセージ |

#### 操作フロー

1. 会議室オーナーが問合せ回答画面を開く
2. 未対応の問合せ一覧が表示される
3. 回答対象の問合せを選択する
4. 問合せ詳細と回答入力欄が表示される
5. 回答内容を入力する
6. 「回答」ボタンを押下する
7. 確認ダイアログで「送信」を押下する
8. 問合せが回答済みに遷移し、一覧が更新される
