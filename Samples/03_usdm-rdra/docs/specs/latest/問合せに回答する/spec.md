# 問合せに回答する

## 概要

会議室オーナーが利用者からの会議室利用に関する問合せに回答し、問合せ状態を「未対応」から「回答済み」に遷移させるユースケース。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 会議室貸出管理フロー | このUCを含むBUC |
| アクター | 会議室オーナー | 操作するアクター |
| 情報 | 問合せ | 更新する問合せの回答内容と状態 |
| 状態 | 問合せ | 未対応→回答済みの状態遷移 |
| バリエーション | 問合せ種別 | オーナー宛問合せ |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 問合せに回答する

  Scenario: 利用者からの問合せに回答する
    Given 会議室オーナー「山田花子」がログイン済み
    And 利用者「田中太郎」からの問合せ「会議室の駐車場の有無について」が「未対応」状態である
    When 会議室オーナー「山田花子」が問合せ回答画面で回答内容「建物に併設の有料駐車場がございます」を入力する
    And 「回答」ボタンを押下する
    Then 問合せ状態が「回答済み」に遷移する
    And 回答日時が記録される
    And 利用者「田中太郎」に回答通知が送信される
```

### 異常系

```gherkin
  Scenario: 回答内容を空のまま回答しようとする
    Given 会議室オーナー「山田花子」がログイン済み
    And 利用者「田中太郎」からの問合せが「未対応」状態である
    When 回答内容を未入力のまま「回答」ボタンを押下する
    Then エラーメッセージ「回答内容を入力してください」が表示される

  Scenario: 既に回答済みの問合せに再度回答しようとする
    Given 会議室オーナー「山田花子」がログイン済み
    And 問合せが既に「回答済み」状態である
    When 当該問合せへの回答を試みる
    Then エラーメッセージ「この問合せは既に回答済みです」が表示される
```
