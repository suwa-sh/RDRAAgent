# 利用者を評価する - フロントエンド仕様

## 変更概要

会議室オーナーが利用終了後の利用者に対して評価を登録する利用者評価登録画面を新規作成する。

## 画面仕様

### 利用者評価登録画面

- **URL**: /owner/ratings/new
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 評価対象一覧 | テーブル | 利用終了済みで未評価の利用一覧（利用者名、会議室名、利用日時） |
| 利用者情報 | テキスト | 評価対象の利用者の氏名 |
| 利用情報 | テキスト | 会議室名、利用開始日時、利用終了日時 |
| 評価スコア | フォーム | 1~5の評価スコア選択 |
| コメント | フォーム | 評価コメント入力テキストエリア |
| 登録ボタン | ボタン | 評価を登録する |

#### 操作フロー

1. 会議室オーナーが利用者評価登録画面を開く
2. 利用終了済みで未評価の利用一覧が表示される
3. 評価対象の利用を選択する
4. 利用者情報と利用情報が表示される
5. 評価スコア（1~5）を選択する
6. コメントを入力する
7. 「登録」ボタンを押下する
8. 評価が登録され、一覧が更新される
