# 利用者の評価を確認する

## 概要

会議室オーナーが予約申請した利用者の過去の利用評価を確認し、使用許諾の判断材料とするユースケース。利用者の評価スコアやコメントを一覧で参照する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 会議室貸出管理フロー | このUCを含むBUC |
| アクター | 会議室オーナー | 操作するアクター |
| 情報 | 利用者評価 | 参照する利用者の評価情報 |
| バリエーション | 評価種別 | 評価の区分（利用者評価） |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 利用者の評価を確認する

  Scenario: 利用者の過去の評価一覧を確認する
    Given 会議室オーナー「山田花子」がログイン済み
    And 利用者「田中太郎」に対する利用者評価が3件登録されている
    When 会議室オーナー「山田花子」が利用者「田中太郎」の評価確認画面を開く
    Then 利用者「田中太郎」の評価が3件表示される
    And 平均評価スコアが表示される

  Scenario: 評価のない利用者の評価を確認する
    Given 会議室オーナー「山田花子」がログイン済み
    And 利用者「新規ユーザー」に対する利用者評価が0件である
    When 会議室オーナー「山田花子」が利用者「新規ユーザー」の評価確認画面を開く
    Then 「評価はまだありません」と表示される
```

### 異常系

```gherkin
  Scenario: 存在しない利用者の評価を確認しようとする
    Given 会議室オーナー「山田花子」がログイン済み
    When 存在しない利用者IDの評価確認画面にアクセスする
    Then エラーメッセージ「指定された利用者が見つかりません」が表示される
```
