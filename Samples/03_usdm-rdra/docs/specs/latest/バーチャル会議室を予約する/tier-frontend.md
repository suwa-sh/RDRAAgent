# バーチャル会議室を予約する - フロントエンド仕様

## 変更概要

バーチャル会議室予約画面を新規作成する。利用者がバーチャル会議室の利用日時を指定して予約申請を行う。物理会議室の予約画面と異なり、バーチャル会議室固有の情報（会議ツール種別・同時接続数）が表示される。

## 画面仕様

### バーチャル会議室予約画面

- **URL**: /rooms/virtual/{room_id}/reserve
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 予約対象のバーチャル会議室名 |
| 会議ツール種別 | テキスト | Zoom/Teams/Google Meet の表示 |
| 同時接続数 | テキスト | 最大同時接続人数の表示 |
| 録画可否 | テキスト | 録画可能かどうかの表示 |
| 価格 | テキスト | 1時間あたり価格の表示 |
| 利用開始日時 | 日時ピッカー | 利用開始日時の選択（必須） |
| 利用終了日時 | 日時ピッカー | 利用終了日時の選択（必須） |
| 予約申請ボタン | ボタン | 予約申請を実行する |

#### 操作フロー

1. 利用者が会議室検索結果または会議室詳細画面からバーチャル会議室予約画面にアクセスする
2. バーチャル会議室の情報（会議ツール種別・同時接続数・録画可否・価格）が表示される
3. 利用開始日時と利用終了日時を選択する
4. 予約申請ボタンを押下する
5. 予約申請完了後、決済方法設定画面へ遷移する
