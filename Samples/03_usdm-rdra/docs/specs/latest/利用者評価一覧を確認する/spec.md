# 利用者評価一覧を確認する

## 概要

会議室オーナーが利用者から受けた評価を一覧で確認し、運営改善に活用するユースケース。オーナー自身の会議室に関する利用者評価を一覧で参照する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 利用実績管理フロー | このUCを含むBUC |
| アクター | 会議室オーナー | 操作するアクター |
| 情報 | 利用者評価 | 参照する利用者の評価情報 |
| バリエーション | 評価種別 | 評価の区分 |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 利用者評価一覧を確認する

  Scenario: 利用者からの評価一覧を確認する
    Given 会議室オーナー「山田花子」がログイン済み
    And 会議室オーナー「山田花子」に対する利用者評価が5件登録されている
    When 会議室オーナー「山田花子」が利用者評価一覧画面を開く
    Then 利用者評価が5件表示される
    And 各評価の評価スコア、コメント、利用者名、評価日時が表示される

  Scenario: 評価がない場合の一覧を確認する
    Given 会議室オーナー「新規オーナー」がログイン済み
    And 会議室オーナー「新規オーナー」に対する利用者評価が0件である
    When 利用者評価一覧画面を開く
    Then 「利用者評価はまだありません」と表示される
```

### 異常系

```gherkin
  Scenario: 未ログインで評価一覧にアクセスする
    Given 未ログイン状態である
    When 利用者評価一覧画面にアクセスする
    Then ログイン画面にリダイレクトされる
```
