# 鍵の返却を記録する - フロントエンド仕様

## 変更概要

会議室オーナーが利用者からの鍵の返却を記録する鍵返却記録画面を新規作成する。

## 画面仕様

### 鍵返却記録画面

- **URL**: /owner/keys/return
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 貸出中鍵一覧 | テーブル | 貸出中の鍵一覧（鍵ID、会議室名、利用者名、貸出日時、利用開始日時、利用終了日時） |
| 鍵情報 | テキスト | 選択した鍵の詳細情報 |
| 会議室利用情報 | テキスト | 対応する会議室利用の状態 |
| 返却記録ボタン | ボタン | 鍵の返却を記録する |
| 確認ダイアログ | テキスト | 返却記録前の確認メッセージ |

#### 操作フロー

1. 会議室オーナーが鍵返却記録画面を開く
2. 貸出中の鍵一覧が表示される
3. 返却対象の鍵を選択する
4. 鍵情報と会議室利用情報が表示される
5. 「返却記録」ボタンを押下する
6. 確認ダイアログで「確定」を押下する
7. 鍵が保管中に遷移し、会議室利用が終了状態となる
