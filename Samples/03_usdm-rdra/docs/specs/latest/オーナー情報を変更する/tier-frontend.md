# オーナー情報を変更する - フロントエンド仕様

## 変更概要

会議室オーナーが自身の登録情報を編集・保存するオーナー情報編集画面を新規作成する。

## 画面仕様

### オーナー情報編集画面

- **URL**: /owners/profile/edit
- **アクセス権**: 会議室オーナー（登録済みのみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| オーナーID | テキスト | オーナーIDを表示（編集不可） |
| 氏名 | テキスト入力 | オーナーの氏名を編集する。必須項目 |
| 連絡先 | テキスト入力 | オーナーの電話番号を編集する。必須項目 |
| メールアドレス | テキスト入力 | オーナーのメールアドレスを編集する。必須項目 |
| 登録日 | テキスト | 登録日を表示（編集不可） |
| 保存ボタン | ボタン | 変更内容を保存する |
| キャンセルボタン | ボタン | 変更を破棄して元の画面に戻る |
| 変更完了メッセージ | テキスト | 保存成功時に表示される完了メッセージ |
| バリデーションエラー | テキスト | 入力不備時に各項目付近に表示されるエラーメッセージ |

#### 操作フロー

1. 会議室オーナーがオーナー情報編集画面にアクセスする
2. 現在のオーナー情報が入力欄に表示される
3. 変更したい項目を編集する
4. 保存ボタンを押す
5. 入力内容のバリデーションが実行される
6. バリデーション成功時、APIにリクエストが送信される
7. 変更完了メッセージが表示される
