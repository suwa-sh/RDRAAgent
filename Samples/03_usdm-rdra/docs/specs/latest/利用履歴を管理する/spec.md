# 利用履歴を管理する

## 概要

サービス運営担当者が会員別・物件別の利用履歴を確認・管理する。利用履歴データの一覧表示、検索、詳細確認を通じてサービスの利用状況を把握する。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 利用状況管理フロー | このUCを含むBUC |
| アクター | サービス運営担当者 | 操作するアクター |
| 情報 | 利用履歴 | 参照・管理する情報（会員別・物件別の利用履歴） |
| バリエーション | 利用履歴集計区分 | 会員別、物件別、期間別 |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: 利用履歴を管理する

  Scenario: 利用履歴一覧を表示する
    Given サービス運営担当者「山田花子」がログイン済み
    And 2026年3月の利用履歴が20件登録されている
    When 利用履歴管理画面にアクセスする
    Then 利用履歴が一覧で表示される
    And 各行に履歴ID・利用者名・会議室名・利用日時・利用時間・利用料金が表示される

  Scenario: 会員別に利用履歴を検索する
    Given サービス運営担当者「山田花子」がログイン済み
    And 利用者「田中太郎」の利用履歴が3件存在する
    When 利用者名「田中太郎」で利用履歴を検索する
    Then 利用者「田中太郎」の利用履歴3件が表示される

  Scenario: 物件別に利用履歴を検索する
    Given サービス運営担当者「山田花子」がログイン済み
    And 会議室「新宿カンファレンスルームA」の利用履歴が5件存在する
    When 会議室名「新宿カンファレンスルームA」で利用履歴を検索する
    Then 会議室「新宿カンファレンスルームA」の利用履歴5件が表示される

  Scenario: 利用履歴の詳細を確認する
    Given サービス運営担当者「山田花子」がログイン済み
    And 利用履歴「H001」が存在する
    When 利用履歴「H001」の詳細を表示する
    Then 履歴ID・利用者ID・会議室ID・利用日時・利用時間・利用料金が表示される
```

### 異常系

```gherkin
  Scenario: 検索結果が0件の場合
    Given サービス運営担当者「山田花子」がログイン済み
    When 存在しない利用者名「存在しないユーザー」で利用履歴を検索する
    Then 「該当する利用履歴がありません」とメッセージが表示される

  Scenario: サービス運営担当者以外がアクセスした場合
    Given 利用者「佐藤次郎」がログイン済み
    When 利用履歴管理画面にアクセスする
    Then 「アクセス権限がありません」とエラーが表示される
```
