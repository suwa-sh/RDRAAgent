# 会議室の詳細を確認する - フロントエンド仕様

## 変更概要

利用者が会議室の物件情報・運用ルール・評価を確認するための会議室詳細画面を新規作成する。

## 画面仕様

### 会議室詳細画面

- **URL**: /rooms/:room_id
- **アクセス権**: 利用者（公開中の会議室のみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 会議室の名称を表示する |
| 画像ギャラリー | 画像 | 会議室の画像をスライドショー形式で表示する |
| 所在地 | テキスト | 会議室の住所を表示する |
| 広さ | テキスト | 平方メートル単位で表示する |
| 収容人数 | テキスト | 会議室の収容可能人数を表示する |
| 価格 | テキスト | 時間あたりの利用料金を表示する |
| 設備・機能一覧 | テーブル | 会議室に備わる設備・機能をアイコン付きで表示する |
| 運用ルール | テーブル | 利用可能時間帯、最低利用時間、最大利用時間、貸出可否を表示する |
| 平均評価スコア | テキスト | 評価スコアの平均値を星表示する |
| 評価コメント一覧 | テーブル | 利用者からの評価コメントを新しい順に表示する |
| 予約申請ボタン | ボタン | 予約申請画面へ遷移する |

#### 操作フロー

1. 利用者が会議室検索結果から会議室を選択する
2. 会議室詳細画面が開き、物件情報が表示される
3. スクロールして運用ルール、評価コメントを確認する
4. 予約を希望する場合、予約申請ボタンをクリックして予約申請画面へ遷移する
