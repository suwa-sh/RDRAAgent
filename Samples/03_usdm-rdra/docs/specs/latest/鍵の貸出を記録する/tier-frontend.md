# 鍵の貸出を記録する - フロントエンド仕様

## 変更概要

会議室オーナーが利用者への鍵の貸出を記録する鍵貸出記録画面を新規作成する。

## 画面仕様

### 鍵貸出記録画面

- **URL**: /owner/keys/lend
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 貸出可能予約一覧 | テーブル | 確定済みで当日利用予定の予約一覧（予約ID、利用者名、会議室名、利用開始日時、利用終了日時） |
| 鍵情報 | テキスト | 対象会議室の鍵ID、現在の貸出状態 |
| 貸出記録ボタン | ボタン | 鍵の貸出を記録する |
| 確認ダイアログ | テキスト | 貸出記録前の確認メッセージ |

#### 操作フロー

1. 会議室オーナーが鍵貸出記録画面を開く
2. 確定済みで当日利用予定の予約一覧が表示される
3. 貸出対象の予約を選択する
4. 鍵情報と利用者情報が表示される
5. 「貸出記録」ボタンを押下する
6. 確認ダイアログで「確定」を押下する
7. 鍵が貸出中に遷移し、会議室利用が開始状態となる
