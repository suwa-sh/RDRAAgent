# バーチャル会議室を登録する - フロントエンド仕様

## 変更概要

バーチャル会議室登録画面を新規作成する。会議室種別別登録条件に基づき、会議室種別「バーチャル」選択時にバーチャル会議室固有の入力項目（会議ツール種別・同時接続数・録画可否・会議URL）を表示する。

## 画面仕様

### バーチャル会議室登録画面

- **URL**: /rooms/virtual/new
- **アクセス権**: 会議室オーナー

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室種別 | セレクトボックス | 「バーチャル」が初期選択。物理・バーチャルの切替で入力項目が動的に変わる |
| 会議室名 | テキスト入力 | 会議室の名称（必須） |
| 会議ツール種別 | セレクトボックス | Zoom、Teams、Google Meet から選択（必須） |
| 同時接続数 | 数値入力 | 最大同時接続人数（必須） |
| 録画可否 | ラジオボタン | 可・不可を選択（必須） |
| 価格 | 数値入力 | 1時間あたり価格（必須） |
| 会議URL | テキスト入力 | 会議ツールのURL（必須） |
| 有効期限 | 日付ピッカー | 会議URLの有効期限（必須） |
| 画像 | ファイルアップロード | 会議室のイメージ画像（任意） |
| 登録ボタン | ボタン | 入力内容を送信して登録する |

#### 操作フロー

1. 会議室オーナーがバーチャル会議室登録画面にアクセスする
2. 会議室種別「バーチャル」が選択された状態で、バーチャル会議室固有の入力項目が表示される
3. 会議室名、会議ツール種別、同時接続数、録画可否、価格を入力する
4. 会議URLと有効期限を入力する
5. 登録ボタンを押下する
6. 登録完了後、会議室管理画面へ遷移する
