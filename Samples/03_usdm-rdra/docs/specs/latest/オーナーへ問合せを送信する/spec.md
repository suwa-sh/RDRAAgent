# オーナーへ問合せを送信する

## 概要

利用者が会議室オーナーに対して会議室利用に関する問合せを行う。問合せ情報が「未対応」状態で作成され、オーナーの回答待ちとなる。会議室貸出管理フロー内の利用者からオーナー宛の問合せである。

## 関連 RDRA モデル

| モデル種別 | 要素名 | 関連 |
|-----------|--------|------|
| BUC | 会議室貸出管理フロー | このUCを含むBUC |
| アクター | 利用者 | 操作するアクター |
| 情報 | 問合せ | 作成する問合せ情報 |
| 状態 | 問合せ | 問合せ状態を「未対応」に遷移 |
| バリエーション | 問合せ種別 | オーナー宛問合せ |

## E2E 完了条件（BDD）

### 正常系

```gherkin
Feature: オーナーへ問合せを送信する

  Scenario: 会議室オーナーへ問合せを送信する
    Given 利用者「田中太郎」がログイン済み
    And 会議室「新宿カンファレンスルームA」のオーナー「佐藤花子」が存在する
    When 会議室「新宿カンファレンスルームA」のオーナーに対して問合せ内容「プロジェクターの使い方を教えてください」を送信する
    Then 問合せが「未対応」状態で作成される
    And 問合せ先区分が「オーナー宛問合せ」である
    And 問合せ送信完了メッセージが表示される
```

### 異常系

```gherkin
  Scenario: 問合せ内容が空で送信しようとする
    Given 利用者「田中太郎」がログイン済み
    When 問合せ内容を空のまま送信しようとする
    Then 「問合せ内容を入力してください」とエラーが表示される
    And 問合せは作成されない
```
