# オーナーへ問合せを送信する - フロントエンド仕様

## 変更概要

問合せ送信画面を新規作成する。利用者が会議室オーナーに対して会議室利用に関する問合せを送信する。

## 画面仕様

### 問合せ送信画面

- **URL**: /rooms/{room_id}/inquiries/new
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 会議室名 | テキスト | 問合せ対象の会議室名を表示 |
| オーナー名 | テキスト | 問合せ先のオーナー名を表示 |
| 問合せ内容 | テキストエリア | 問合せ内容を入力する |
| 送信ボタン | ボタン | 問合せを送信する |
| エラーメッセージ | テキスト | バリデーションエラー時に表示 |

#### 操作フロー

1. 利用者が会議室詳細画面から問合せ送信画面に遷移する
2. 会議室名とオーナー名が表示される
3. 問合せ内容を入力する
4. 送信ボタンを押下する
5. 問合せ送信完了メッセージが表示される
