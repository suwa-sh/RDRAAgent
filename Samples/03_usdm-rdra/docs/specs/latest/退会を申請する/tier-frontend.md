# 退会を申請する - フロントエンド仕様

## 変更概要

会議室オーナーがサービスからの退会を申請するための退会申請画面を新規作成する。

## 画面仕様

### 退会申請画面

- **URL**: /owners/withdraw
- **アクセス権**: 会議室オーナー（登録済みのみ）

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 退会注意事項 | テキスト | 退会による影響（公開中の会議室が非公開になること等）を説明する |
| 退会理由 | テキストエリア | 退会理由を任意で入力する |
| 退会申請ボタン | ボタン | 退会申請を実行する |
| キャンセルボタン | ボタン | 退会を中止して元の画面に戻る |
| 確認ダイアログ | ダイアログ | 退会申請前の最終確認を表示する |
| 退会完了メッセージ | テキスト | 退会完了時に表示されるメッセージ |
| エラーメッセージ | テキスト | 未精算等で退会不可の場合に表示されるメッセージ |

#### 操作フロー

1. 会議室オーナーが退会申請画面にアクセスする
2. 退会による影響に関する注意事項を確認する
3. 必要に応じて退会理由を入力する
4. 退会申請ボタンを押す
5. 確認ダイアログが表示される
6. 確認ダイアログで「はい」を押すとAPIにリクエストが送信される
7. 退会完了メッセージが表示される
