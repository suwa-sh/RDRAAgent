# サービスへ問合せする - フロントエンド仕様

## 変更概要

サービス問合せ画面を新規作成する。利用者がサービス運営担当者宛の問合せ内容を入力・送信できるようにする。

## 画面仕様

### サービス問合せ画面

- **URL**: /inquiry/service
- **アクセス権**: 利用者

#### 表示要素

| 要素 | 種別 | 説明 |
|------|------|------|
| 問合せ内容 | テキストエリア | 問合せ内容を入力（必須） |
| 送信ボタン | ボタン | 問合せを送信する |
| 問合せ履歴一覧 | テーブル | 過去の問合せ一覧（問合せID、問合せ内容（先頭50文字）、問合せ状態、問合せ日時） |
| 完了メッセージ | テキスト | 送信完了時のメッセージ |

#### 操作フロー

1. 利用者がサービス問合せ画面にアクセスする
2. 問合せ内容を入力する
3. 送信ボタンを押下する
4. 問合せが送信され完了メッセージが表示される
5. 問合せ履歴一覧に送信した問合せが追加される
