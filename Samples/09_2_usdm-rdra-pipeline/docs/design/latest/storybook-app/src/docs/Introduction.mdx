## 技術スタック

- **フレームワーク**: Next.js + TypeScript
- **スタイリング**: Tailwind CSS v4 + CSS Variables (Design Tokens)
- **コンポーネント**: React Function Components
- **ドキュメント**: Storybook + MDX

## コンポーネント構成

### UI コンポーネント (汎用)
- **Button** - 全アクション (5 variants x 3 sizes)
- **Badge** - ステータス表示 (6 variants)
- **Card** - コンテンツコンテナ (hoverable)
- **Input** - フォーム入力 (label, error state)
- **Icon** - SVG アイコン (20 icons)

### ドメインコンポーネント
- **RoomCard** - 会議室情報カード
- **SearchFilter** - 検索フィルター
- **BookingCalendar** - 予約カレンダー
- **ReservationCard** - 予約情報カード
- **StarRating** - 星評価 (SVG)
- **StatusBadge** - ステータスバッジ (5 状態モデル対応)
- **StepTracker** - ステップトラッカー
- **PriceDisplay** - 価格表示
- **InquiryThread** - 問合せスレッド
- **SummaryCard** - 集計サマリーカード
