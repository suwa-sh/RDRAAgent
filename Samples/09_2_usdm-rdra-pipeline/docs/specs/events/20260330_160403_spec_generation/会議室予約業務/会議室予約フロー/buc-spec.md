# 会議室予約フロー

## 概要

利用者が条件で会議室を検索し、予約と決済方法を設定するフロー。

## 所属 UC 一覧

| UC名 | アクター | 主な操作 | 関連情報 |
|------|---------|---------|---------|
| [会議室を照会する](会議室を照会する/spec.md) | 利用者 | 会議室の検索・一覧表示 | 会議室情報, 会議室評価 |
| [会議室を予約する](会議室を予約する/spec.md) | 利用者 | 会議室の予約・決済方法設定 | 予約情報, 決済情報 |
