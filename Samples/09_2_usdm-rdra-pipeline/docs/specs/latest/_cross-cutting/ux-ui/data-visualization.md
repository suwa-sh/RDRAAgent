## チャート選定ガイドライン

### 観点別チャート選定

| 観点 | 推奨チャート | 使用場面 |
|------|-----------|---------|
| Comparison（比較） | 横棒グラフ | 会議室別売上、会議室別手数料、物件別利用率。カテゴリ名が長い日本語のため横棒が適切 |
| Composition（構成比） | 積み上げ棒グラフ | 会議室別売上構成。5項目以上になる可能性があるため円グラフではなく積み上げ棒 |
| Distribution（分布） | ヒストグラム | 利用時間分布、手数料率分布。連続値の分布を把握する |
| Trend（傾向） | 折れ線グラフ、エリアチャート | 月次売上推移、手数料推移、利用件数推移。時系列の変化を追跡 |

### データ表現フレームワーク

| 表現方法 | 用途 | 適用画面 |
|---------|------|---------|
| % Change（変化率） | 前月比の成長率表示 | SummaryCardの trend プロパティで表示 |
| Variance（差異） | 目標値との差分表示 | 手数料売上分析の KPI |
| Calculated Metric（計算指標） | 会議室あたり売上、利用者あたり利用回数 | 利用実績確認、利用状況分析 |
| Added Context（追加コンテキスト） | 前年同月ラインの追加 | 売上推移グラフ |

## ダッシュボード設計原則

### 情報の階層化

- **全体サマリー**: SummaryCard 4枚で KPI 概要（利用件数、売上、手数料、前月比）を一覧表示
- **ドリルダウン**: SummaryCard クリックで詳細チャートを展開。折りたたみ可能なセクション
- **フィルター**: 期間（月/四半期/年）、会議室、オーナーでの絞り込み。フィルター変更は全チャートに連動

### データストーリーテリング

- **ナラティブ**: 各ダッシュボードに「今月のサマリー」テキストを自動生成（前月比の傾向を言語化）
- **比較軸**: 前月比、前年同月比を SummaryCard の change/trend で表示
- **アクション**: 売上が減少傾向の会議室を強調表示し、運用ルール見直しを促す

## 認知負荷への配慮

- ワーキングメモリの限界（4-5項目）を考慮し、ダッシュボード上段の SummaryCard は最大4枚
- Data-Ink Ratio を意識し、チャートのグリッド線を最小限に。背景色は semantic.muted (#F1F5F9)
- ゲシュタルトの近接法則に基づき、関連指標を Card 内にグルーピング
- チャートのカラーパレットは design-event.yaml の primitive colors から選定（blue-500, teal-500, amber-500, green-500, red-500 の5色以内）
