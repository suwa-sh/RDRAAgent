# RDRASpreadsheetに展開

## 目的
RDRAスプレッドシートにRDRAモデルデータをコピーします

## 実行方法
ターミナルで以下の手順を実行してください:

### MacOS

```bash
node RDRA_Knowledge/helper_tools/makeZeroOneData.js
node RDRA_Knowledge/helper_tools/copyToClipboard.js zeroone
open "https://docs.google.com/spreadsheets/d/1h7J70l6DyXcuG0FKYqIpXXfdvsaqjdVFwc6jQXSh9fM/edit?gid=1240873646#gid=1240873646"
```

### Linux

```bash
node RDRA_Knowledge/helper_tools/makeZeroOneData.js
node RDRA_Knowledge/helper_tools/copyToClipboard.js zeroone
xdg-open https://docs.google.com/spreadsheets/d/1h7J70l6DyXcuG0FKYqIpXXfdvsaqjdVFwc6jQXSh9fM/edit?gid=1240873646#gid=1240873646"
```

### Windows

```cmd
node RDRA_Knowledge/helper_tools/makeZeroOneData.js
node RDRA_Knowledge/helper_tools/copyToClipboard.js zeroone
start https://docs.google.com/spreadsheets/d/1h7J70l6DyXcuG0FKYqIpXXfdvsaqjdVFwc6jQXSh9fM/edit?gid=1240873646#gid=1240873646
```

> **注意**: 日本語ファイル名の文字化け問題を回避するため、`zeroone` キーワードを使用しています。

### ファイル生成が完了したら、要約せずに停止してください
